# About

## What

This is a personal iOS project I made for checking STM bus arrivals in Montreal 🚌

It helps me see nearby buses, search routes or stops, and check live bus information in a simpler way

## Why

Around the quarter of 2025, I started using STM transportation again

During 2025, the STM workers strike happened which led to limited bus hours and fewer buses passing by some stops 😩 because of that, it became harder for me to know when the next bus would arrive at the stop where I needed to take it 😮‍💨

That was the main reason I got motivated to create this app 💡 I wanted something for personal use where I could at least see live buses near my location and check arrivals more easily

Right now, I am also convincing my partner to use it whenever she needs to check the bus and give me feedback. I want to improve it more and hopefully upload it to the App Store soon 🤞

One big reason I keep working on it is because I want something that feels easy and quick for Montrealers like me. With apps like Google Maps, Chrono, or Transit, sometimes it feels like too much tapping just to get the one thing I need 🤯

I want this app to be simple, fast, and direct. Open it, check the bus and move on ✨

Later on, I decided to push the code to GitHub 🤓

## How

The app uses STM open transit data

It loads static schedule data and combines it with live STM data like vehicle positions, trip updates, and service alerts. Then it shows that information through nearby arrivals, search, favorites, warning notices and live map details

I built it with SwiftUI and made it in a way that it can still show scheduled information when live data is not available. The goal is to keep the experience easy, useful, and quick 🚏🏃
